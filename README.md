# TechStore - Página Web de Tienda de Tecnología

Bienvenido a **TechStore**, una página web completa de una tienda de tecnología desarrollada con **HTML, CSS y JavaScript**.

## 📋 Descripción

TechStore es una página web moderna y profesional que ofrece:

- 🏠 **Página de Inicio**: Bienvenida e introducción al negocio
- 🛍️ **Catálogo de Productos**: Listado de 6 productos tecnológicos
- 📍 **Página de Ubicación**: Dirección física, mapa de Google Maps
- 📧 **Página de Contacto**: Formulario de contacto y redes sociales
- 💳 **Comercio Electrónico**: Carrito de compras y múltiples métodos de pago
- 🎥 **Video de YouTube**: Integrado en la página de comercio electrónico

## 🚀 Características

### Página de Inicio (index.html)
- Logo y nombre del negocio (TechStore)
- Bienvenida e introducción
- Descripción general del negocio
- Botón de llamada a la acción

### Catálogo (catalogo.html)
- Listado de 6 productos tecnológicos
- Precio de cada producto
- Descripción detallada
- Botones para comprar

### Ubicación (ubicacion.html)
- Dirección física real
- Mapa embebido de Google Maps
- Información de referencia
- Horario de atención

### Contacto (contacto.html)
- Formulario de contacto funcional
- Correo electrónico de contacto
- Número telefónico
- Enlaces a redes sociales

### Comercio Electrónico (ecommerce.html)
- Carrito de compras interactivo
- Formulario de compra completo
- Métodos de pago: Transferencia, Tarjeta, PayPal
- Video de YouTube integrado
